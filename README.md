# FlagTip 🚩  

FlagTip은 현재 입력 중인 위치를 추적하여  
현재 언어 상태를 깃발로 출력하는 프로그램입니다.


---

## 🧠 왜 만들었나요?

내가 타이핑할 때 한글 혹은 영문인지 알기위해 제작하였습니다.


## 📦 설치

### MSIX (권장)

1. 아래 AppInstaller 파일을 다운로드
2. 더블 클릭 후 설치


---

## 🔄 업데이트

- AppInstaller 기반 자동 업데이트 지원
- 새 버전 배포 시 재부팅 또는 재실행 시 자동 체크
