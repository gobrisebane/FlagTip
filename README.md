## 크롬에서 깃발이 제대로 반영 안되는 현상
해당 에러는 새로 업데이트된 윈도우기능의 문제이므로 해당 기능을 이전모드로 바꿔주셔야 합니다. 해당 모드는 이전모드로 바꾸어도 윈도우 이용에 문제 없습니다.

1번 방법)

오른쪽 하단 가 혹은 A로 표시된 부분 오른쪽 클릭
-> 설정 클릭 -> 이전 버전의 Microsoft Ime 켜기

2번 방법)

윈도우 설정 -> 시간 및 언어 -> 언어 및 지역 -> 한국어 오른쪽 설정(...)클릭 -> 언어 옵션 클릭 -> Microsft 입력기 오른쪽 설정(...) 클릭 -> 키보드옵션 -> 이전 버전의 Microsoft Ime 켜기
